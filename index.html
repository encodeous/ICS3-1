Answer to Questions...

  1. What does HTML stand for?

    HTML stands for the Hyper Text Markup Language. It was initially released in 1993, and was standardized by the W3C. For more information, you can check out this Wikipedia page!


  2. What are Markup Tags?
    <h1> Cool! </h1>

    Markup tags are the foundation of HTML, they tell the browser how to render content on a webpage. Different tags have different actions. For example, an h1 tag will make the text contain in it the size of a title.

  3. What tools are required to make a webpage?

    Creating a webpage requires very basic tools. The only tools needed are a text-editor and a browser to view the website.

  4. What extension must your webpage have?

    Webpages, specifically HTML documents must end in .html/.htm to be viewed by the browser.

  5. What is an HTML Element?

    HTML elements are elements defined by one or more tags. Some elements include content which require a starting and closing tag, while others do not.

    Examples:

    Image elements do not contain any content, so it is a single tag.
    <img alt="This is an image" src="image-link-here">

    Paragraphs require closing tags
    <p> Here is some text </p>

  6. What is a nested HTML element?
    Here is a centered text in a wrapper. (H6)

    Here is a nested h3 element

    Items can be nested in html. This allows for much more flexibility when making webpages.

  7. What is an attribute?
    <p class="fancy-text"> Here is some text </p>

    Attributes provide tags with additional information. Some of the most common attributes are style and class, which can apply css to a specific element.

  8. Should tags be uppercase or lowercase?

    In HTML, uppercase tags behave the same way as lowercase ones. They are identical. However, it is generally accepted that tags should be in lowercase. This improves readability and ensures compatibility across browsers.

  9. Give 2 examples of empty HTML elements.

    1. The break-line tag <br>

    2. The horizontal rule tag <hr>

  10. Why is it important to use Heading Tags?

    Heading tags ensure the readability of the document. They stand out from regular text, and tells the reader that a piece of text is a title.

  11. What are comments used for?

    In software development, comments are used to ensure that people can understand what some piece of code is. In HTML, these comments can ensure other designers can interpret what a piece of markup is doing.

  12. How can you see the HTML code for a web page that someone else made and posted on the internet?

    HTML code can be viewed by nearly all text editors by just opening the .html file with it. To preview / render the document, just open it with a browser.

  13. What does deprecated mean? Why should the font tag not be used? What is the correct way to change font?

    As software change over time, there are features that are no longer supported. To ensure legacy code compatibility, software developers often deprecate features so that old software do not break. In this case, the font tag is not longer supported, an alternative is to use CSS properties like font, font-family, color, and font-size to change the text style.

  14. What are 3 ways you can style your HTML code?
    • HTML code can be styled with CSS

      The most basic way to style html element is using the <style> tag. Developers are able to define styles for tags, or create classes that can be used with the class attribute.

    • HTML code can be styled with HTML Tags

      There are a few HTML tags that come with their own styles. A good example is the title tag.

    • HTML code can be styled with inline CSS styles

      HTML elements can also be styled inline, meaning the style is declared along with the tag. This is useful when a style is only used once throughout a document. It can be achieved by using the style attribute.

  15. Why is it not really a good idea to use the style attribute to change the way your web pages look? What is the best method of changing the style of webpages?

    Styling elements with the style attribute can lead to unmaintainable and hard-to-read code. Extensive use of this attribute causes a lot of duplicated code. A simple solution to this is to use the style tag, which allows you to define classes which can be used by multiple tags without duplicating code.

  16. What is the target attribute used for in the hyperlink tag? Give an example.

    The target attribute can be used on anchor tags. This attribute specifies where a link will open. Using the _blank parameter will open the link in a new tab, while using the _top parameter will open the link in the current tab.
